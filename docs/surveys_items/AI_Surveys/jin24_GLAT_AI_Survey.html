Surveys and Questionnaires from Jin et al. (2024)

Jin, Y., Martinez-Maldonado, R., Gašević, D., & Yan, L. (2024). GLAT: The Generative AI Literacy Assessment Test (No. arXiv:2411.00283). arXiv. https://doi.org/10.48550/arXiv.2411.00283

GLAT: The Generative AI Literacy Assessment Test (Initial 25-Item Version)

This is the initial 25-item version presented in Table 2 of the paper, used for content validity assessment and item selection via Classical Test Theory. Items subsequently dropped for the final 20-item version (used in IRT and external validity analysis) are marked. Correct answers are indicated in bold.

  1. Question: Which of the following best describes “Generative AI”?
    1. AI that creates new content like text, images, or music by learning from existing data.
    2. An AI system designed to enhance the speed and accuracy of data retrieval in search engines.
    3. A form of artificial intelligence that focuses on translating languages in real-time.
    4. AI technology used primarily for managing and organizing large databases.
  2. Question: Which of the following statements best describes an LLM (Large Language Model)?
    1. It generates text by analyzing and summarizing large volumes of web content.
    2. It generates text by predicting the next word based on the context of previous words.
    3. It generates text by translating input text into multiple languages simultaneously.
    4. It generates text by using pre-defined templates and filling in the blanks.
  3. Question: Which of the following tasks can Generative AI perform with a high degree of accuracy?
    1. Predicting stock market trends
    2. Making ethical decisions in complex scenarios
    3. Diagnosing rare diseases
    4. Generating human-like text based on prompts
  4. Question: In the context of Generative AI, what is “zero-shot learning”?
    1. Training a model without any data.
    2. The ability of a model to perform a task without any task-specific training.
    3. A method of reducing the model’s training time to zero.
    4. A technique for generating synthetic training data.
  5. Question: Which of the following is a potential challenge when using prompt-based development for text generation?
    1. The language model can only generate binary outputs.
    2. The need for extensive labeled data to train the model.
    3. Crafting a prompt that accurately captures the desired context and nuances.
    4. The requirement for complex feature engineering.
  6. Question: [DROPPED ITEM] When using a generative AI model to classify text into multiple categories, what is a common approach to handle more than two output classes?
    1. Use multiple binary classifiers for each category.
    2. Use a single prompt that includes all possible categories.
    3. Train a separate model for each category.
    4. Use unsupervised learning to cluster the text data.
  7. Question: What does the term “token” refer to in the context of a large language model (LLM)?
    1. A token is a unit of text, such as a word or a subword, that the model processes individually.
    2. A token is a unique identifier assigned to each user interacting with the language model.
    3. A token is a security measure used to authenticate API requests to the language model.
    4. A token is a reward given to users for contributing valuable data to train the language model.
  8. Question: Which of the following is NOT a requirement for an AI to be considered artificial general intelligence (AGI)?
    1. The ability to learn and adapt to new tasks without human intervention.
    2. The capability to perform tasks across various domains with human-like proficiency.
    3. The ability to predict future events with perfect accuracy.
    4. The capacity to understand and generate natural language.
  9. Question: [DROPPED ITEM] Why is model alignment important in the development of generative AI?
    1. To ensure AI systems better reflect human values and are safer.
    2. To improve computational efficiency and reduce energy consumption.
    3. To enhance the alignment between model responses and user requests.
    4. To increase the speed of data processing and analysis.
  10. Question: How does RAG (Retrieval-Augmented Generation) enhance the capabilities of an LLM?
    1. By improving its grammar and syntax.
    2. By providing it with real-time and relevant data.
    3. By increasing its computational speed.
    4. By enabling it to understand multiple languages.
  11. Question: When using generative AI to create a marketing pitch, which of the following strategies is least likely to be effective?
    1. Supplying the AI with information about the target audience
    2. Asking the AI to include unique selling points and benefits
    3. Requesting the AI to use persuasive language techniques
    4. Providing the AI with a list of competitors’ products
  12. Question: After deploying a customer service chatbot, you notice that it frequently provides outdated information about company policies. What is the best course of action to address this issue?
    1. Implement a feedback loop where users can flag outdated information for review.
    2. Schedule regular updates to the chatbot’s training data to include the latest company policies.
    3. Set up a system where complex or policy-related queries are escalated to human agents for accurate responses.
    4. Conduct a comprehensive audit of the chatbot’s performance metrics to identify areas for improvement.
  13. Question: Suppose you have a large dataset of emails and you want to build an application to answer questions based on this dataset. Which of the following scenarios best illustrates the advantage of using RAG over prompting (i.e., without RAG)?
    1. You need to generate creative writing pieces based on the email content.
    2. You want to ensure the model can answer questions even if it has never seen similar questions before.
    3. You need to answer questions that require specific information from different parts of the email dataset.
    4. You want to reduce the size of the language model to save computational resources.
  14. Question: [DROPPED ITEM] While using a Generative AI tool to write a story, you notice that the context window is limited to 500 tokens. What is a potential consequence of exceeding this limit?
    1. The AI will automatically expand the context window
    2. The AI will ignore the excess tokens and generate text based on the first 500 tokens
    3. The AI will generate text based on the most recent 500 tokens
    4. The AI will stop functioning until the context window is reduced
  15. Question: [DROPPED ITEM] When creating a video with a generative AI tool that supports text, images, and audio narration, which feature is most critical for ensuring the tool can handle this task effectively?
    1. Text-to-Speech (TTS) capability.
    2. Image recognition capability.
    3. Multilingual support.
    4. Sentiment analysis capability.
  16. Question: As a student using a Large Language Model (LLM) to gather information for an assignment, how should you approach the information it provides?
    1. The LLM’s answers are always more trustworthy than any information you will find on the internet, so you can use them without further verification.
    2. The LLM’s answers are generally more trustworthy than internet sources, but you should still verify the information with other reliable sources.
    3. The LLM’s answers are not necessarily more trustworthy than internet sources, and you should cross-check the information with other credible references.
    4. The LLM’s answers are less trustworthy than internet sources because it relies on outdated information.
  17. Question: It is unlikely for an LLM to provide an accurate summary of the latest financial market trends in real-time. Is this statement true or false?
    1. True, because the LLM’s data may be outdated due to its knowledge cutoff.
    2. True, because the LLM is not good at handling numbers and structured data.
    3. False, because the LLM frequently updates its knowledge base.
    4. False, because the LLM is capable of synthesizing the latest market data automatically.
  18. Question: A generative AI tool has provided a summary of a research paper. The summary states, “The study found that increased screen time is directly correlated with decreased attention spans in children aged 8-12.” What is your next step?
    1. Accept the summary as accurate because AI tools are generally reliable.
    2. Ask the AI to provide more details about the study’s methodology and results.
    3. Cross-check the summary with the original research paper.
    4. Use another AI tool to generate a summary for comparison and evaluate the consistency between both summaries
  19. Question: While reviewing a video of a well-known public figure making controversial statements, which characteristic confirms the video was NOT generated by AI?
    1. The public figure’s voice sounds like themselves.
    2. The video has a professional and polished appearance.
    3. The video is high-quality with smooth transitions.
    4. None of the above.
  20. Question: [DROPPED ITEM] Imagine someone is trying to clone your voice using generative AI. Which of the following audio recordings would be most effective?
    1. A 1-hour recording of your conversation in a noisy public space.
    2. A 30-second recording of your presentation speech.
    3. A 5-minute recording of you singing a pop song.
    4. Generative AI cannot be used to clone human voice.
  21. Question: When a generative AI system is used for screening job applications, what issue might arise concerning the quality and fairness of hiring decisions?
    1. The AI system might overlook applicants’ unique achievements and extracurricular activities.
    2. The AI system could misinterpret minor formatting differences in resumes.
    3. The AI system might not effectively handle applications submitted in various languages.
    4. The AI system could reinforce existing biases found in historical hiring data.
  22. Question: In a healthcare startup, an accurate AI model recommends treatments, but doctors don’t trust it because they can’t understand how the model arrived at its conclusions. What core issue does this scenario illustrate?
    1. The AI model uses obsolete training data.
    2. The training dataset lacks sufficient diversity.
    3. The treatment guidelines input are incorrect.
    4. The AI model behaves as a black box.
  23. Question: What are the potential copyright implications for a journalist using an AI-generated image in a commercial article?
    1. The journalist needs to check the licensing policy of the AI tool they used.
    2. The AI-generated image is automatically free to use without any restrictions.
    3. The journalist must pay a standard licensing fee to use the AI-generated image.
    4. The image cannot be used in any commercial context because it is AI-generated.
  24. Question: Should we impose restrictions on the outputs of generative AI technologies?
    1. Yes, to reduce the computational resources required for operating these technologies.
    2. Yes, to prevent the dissemination of harmful or misleading content.
    3. No, as it would hinder technological innovation and creativity.
    4. No, because users should have the freedom to access all generated content.
  25. Question: Sending personal information to cloud-based generative AI tools has little privacy concerns.
    1. True, as this information is encrypted using sophisticated algorithms during the transmission process.
    2. True, as generative AI tools are black-box systems and cannot output personal information even if it is used for model training.
    3. False, as generative AI tools train on unencrypted data and can output private information based on their probabilistic nature.
    4. False, as advancements in quantum computing can easily decipher the encrypted data.

GLAT Content Validity Questionnaire

This questionnaire (Table 3) was used in the pilot study (N=200) to assess the content validity of the initial GLAT items. Participants responded on a 5-point Likert scale (1 = Strongly Disagree to 5 = Strongly Agree). Mean and SD are reported in the paper but the scale itself is presented here.

  1. The questions are directly related to generative AI concepts and skills. (Scale: 1=Strongly Disagree, 2=Disagree, 3=Neutral, 4=Agree, 5=Strongly Agree)
  2. The test covers a broad range of concepts that are necessary for assessing generative AI literacy. (Scale: 1=Strongly Disagree, 2=Disagree, 3=Neutral, 4=Agree, 5=Strongly Agree)
  3. The test includes questions that assess both fundamental and advanced concepts. (Scale: 1=Strongly Disagree, 2=Disagree, 3=Neutral, 4=Agree, 5=Strongly Agree)
  4. The questions are clearly written and easy to understand. (Scale: 1=Strongly Disagree, 2=Disagree, 3=Neutral, 4=Agree, 5=Strongly Agree)
  5. The options provided for each question are clearly distinct and easily distinguishable. (Scale: 1=Strongly Disagree, 2=Disagree, 3=Neutral, 4=Agree, 5=Strongly Agree)
  6. Overall, I believe the test is an effective tool for assessing generative AI literacy. (Scale: 1=Strongly Disagree, 2=Disagree, 3=Neutral, 4=Agree, 5=Strongly Agree)

External Validity Task - Example Questions (Bar Chart)

These are example questions (Table 4) related to the bar chart visualization used in the external validity learning task (Section 3.4.1). The task involved analyzing three visualizations (bar chart, social network, heatmap) and answering six multiple-choice questions (two per visualization) targeting Bloom’s knowledge and comprehension levels.

Bloom’s Level Question
Knowledge Which behaviour did the two nurses spend the most time on?
Comprehension How did the nurses spend their time working on tasks for Amy compared to other tasks?

Other Instruments Mentioned (Not Reproduced in Paper)

The study also utilized the following instruments during the external validity phase (Section 3.4.3), but their specific items are not reproduced in the Jin et al. (2024) paper:

  • ChatGPT Literacy Scale: (Lee & Park, 2024) - A self-report measure of GenAI literacy, specifically related to ChatGPT.
  • Mini-VLAT: (Pandey & Ottley, 2023) - A measure of visualization literacy (domain knowledge for the external validity task).
